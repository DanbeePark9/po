SUMMARY

어릴때부터 호기심이 많아 웹게임에서 HTML 버그를 찾고 놀던 아이가 어느새 자라 웹개발자가 되었습니다. 영어, 일본어, 중국어, 프랑스어 등 다개국어를 능숙하게 구사 할 수 있을 만큼 외국어에 열정이 있었던 저는 외국어 관련 어플을 직접 만들고자 Xcode, Android Studio 를 설치하여 독학으로 개발을 시작했었습니다.

퀄리티 높은 어플, 웹 서비스를 만들고 싶은 욕심이 생기게 되어 JAVA 앱, 웹개발자 과정을 수료하게 되었고, 평생 교육원을 통해 컴퓨터공학 학사 과정을 진행하며 일과 학업을 병행하고 있습니다.
